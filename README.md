# README – Valoración de la Práctica UD3 (Apache Tomcat)

## Grado de complejidad

El grado de complejidad de la práctica no es muy alto. Aunque incluye bastantes pasos y configuraciones (instalación, despliegue, securización e integración con un IDE), el gran número de paginas vinculadas y documentación presente en la web lo hace más llevadero.

Algunas partes, como la configuración de SSL o la creación del servicio systemd en Linux, requieren mayor atención y cuidado, pero no resultan excesivamente complejas si se siguen los pasos correctamente.

## Valoración personal y comprensión

En general, los ejercicios me parecieron **asequibles**, especialmente una vez entendido el funcionamiento básico de Apache Tomcat. La práctica me ayudó a comprender mejor:

- Cómo funciona Tomcat como servidor de aplicaciones.
- El proceso de despliegue de aplicaciones web mediante archivos WAR.
- La diferencia entre usar Tomcat en Windows (con XAMPP) y en Linux.
- Aspectos básicos de administración, seguridad (HTTPS) y monitorización.

Tras realizar la práctica, considero que he entendido el funcionamiento general de los servidores Apache Tomcat y su uso en un entorno real de despliegue de aplicaciones web.
